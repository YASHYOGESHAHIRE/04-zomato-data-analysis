# Zomato Data Analysis using Python

This project performs Exploratory Data Analysis (EDA) on Zomato restaurant data using Python. The goal is to uncover insights about restaurant trends, customer preferences, cuisines, ratings, pricing, and online ordering behavior.

The project demonstrates how data analysis and visualization techniques can be applied to real-world business datasets.

---

# Overview

Zomato is a popular restaurant discovery and food delivery platform. This dataset contains information about restaurants, cuisines, ratings, pricing, votes, and services offered.

This project focuses on:

- Cleaning and preprocessing data
- Exploring restaurant trends
- Analyzing customer ratings
- Understanding pricing patterns
- Visualizing key business insights

---

# Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Restaurant rating analysis
- Cuisine trend analysis
- Online order and table booking analysis
- Cost distribution analysis
- Data visualization using charts and graphs

---

# Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

# Dataset Insights

The analysis explores:

- Most popular restaurant types
- Top-rated cuisines
- Average cost for two people
- Online ordering trends
- Restaurant location analysis
- Relationship between ratings and votes
- Price vs rating comparisons

---

# Project Workflow

1. Import dataset
2. Clean and preprocess data
3. Handle missing values
4. Perform exploratory analysis
5. Generate visualizations
6. Extract business insights
